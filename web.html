En esta página web encontrarás conceptos básicos de HTML, CSS y JS para empezar a crear páginas web.

HTML

HTML es el lenguaje de marcado estándar para la creación de páginas web. Aquí encontrarás conceptos básicos de HTML, como estructura de página, etiquetas básicas, enlaces y anclas, formularios, tablas y listas.

Ver más

CSS

CSS es el lenguaje de estilo estándar para la creación de diseños web. Aquí encontrarás conceptos básicos de CSS, como selectores y propiedades, colores y fondos, tipografía y texto, layout y diseño, pseudoclases y pseudoelementos, y media queries.

Ver más

JS

JS es el lenguaje de programación estándar para la creación de interactividad web. Aquí encontrarás conceptos básicos de JS, como variables y tipos de datos, funciones, condicionales y bucles, eventos, y interacción con el DOM.
